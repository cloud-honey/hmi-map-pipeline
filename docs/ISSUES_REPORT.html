#1 np.ndarray에 .save() 없음

ISSUE-001 RuntimeError output_qa.py → main_pipeline.py
상황ISO 렌더러가 np.ndarray 반환 → PIL.Image 아니므로 .save() 호출 불가
발생main_pipeline.py:152 — det_img.save(det_path)
에러AttributeError: 'numpy.ndarray' object has no attribute 'save'
원인ISORenderer.render() → numpy array 반환인데 PIL.Image로 가정하고 .save() 호출
해결: PIL.Image.fromarray(outputs["render"]) 변환 후 save 
# Before
outputs = renderer.render(self.geometry)
det_img = outputs["render"]
det_img.save(det_path)

# After
det_img_arr = outputs["render"]
det_img = PILImage.fromarray(det_img_arr)
det_img.save(det_path)

ISSUE-002 RuntimeError output_qa.py
상황export_all이 output_dir에 직접 저장 → generate()에서 같은 경로로 copy 시도
발생output_qa.py:373 — shutil.copy(masks_path, masks_dest)

#3 numpy bool JSON 직렬화 실패

ISSUE-003 TypeError output_qa.py — QAReport.to_dict()
상황numpy bool/float/int → Python native type 변환 없음
발생output_qa.py:396 — json.dump(qa_report.to_dict(), f)
에러TypeError: Object of type bool_ is not JSON serializable
원인QAReport.passed가 numpy.bool_ 타입 → json.dumps가 이를 처리 못함
해결: to_dict() 에 _native() helper 추가 
def _native(obj):
    if isinstance(obj, (np.bool_,)):
        return bool(obj)
    if isinstance(obj, (np.integer,)):
        return int(obj)
    if isinstance(obj, (np.floating,)):
        return float(obj)
    return obj

#4 np.maximum deprecation 경고

ISSUE-004 DeprecationWarning output_qa.py — check_color_rules()
상황np.maximum에 3개 이상의 인자 전달 → deprecated
발생output_qa.py:181 — np.maximum(a, b, c)
경고Passing more than 2 positional arguments to np.maximum is deprecated
원인grayscale deviation 계산 위해 3개 배열의 element-wise maximum 필요
해결: np.maximum.reduce([a, b, c]) 사용 
# Before
grayscale_deviation = np.maximum(
    np.abs(r.astype(int) - g.astype(int)),
    np.abs(g.astype(int) - b.astype(int)),
    np.abs(b.astype(int) - r.astype(int))
)

# After
grayscale_deviation = np.maximum.reduce([
    np.abs(r.astype(int) - g.astype(int)),
    np.abs(g.astype(int) - b.astype(int)),
    np.abs(b.astype(int) - r.astype(int))
])

핵심 설계 결정

설계 결정론적 렌더링 우선 — AI는 시각적 보완

구조(wall/room/column) → deterministically rendered (AI 환상 없음)
시각적 품질(텍스처/조명) → AI-enhanced (denoise 0.25~0.35)

설계 타일드 렌더링 — VRAM 8GB 제약 대응

512×512 또는 768×768 타일 + gaussian blending
4K+ 출력도 VRAM 8GB에서 처리 가능

설계 QA 폴백 — AI 실패 시 deterministic 자동 전환

overall < 0.60 또는 align < 0.50 → deterministic render 자동 사용
실패 없음: 항상 유효한 background.png 출력
